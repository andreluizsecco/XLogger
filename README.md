## Installing / Getting started

This package is available through Nuget: [https://www.nuget.org/packages/XLogger](https://www.nuget.org/packages/XLogger)

#### Nuget
```
Install-Package XLogger
```

#### .NET CLI
```
dotnet add package XLogger
```
